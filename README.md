# React API Users

Projeto simples em React consumindo uma API e exibindo usuários na tela.

## 🚀 Tecnologias

- React
- JavaScript
- CSS
- Fetch API

## 📌 Funcionalidades

- Buscar usuários de uma API
- Listar nome, username e email
- Renderização dinâmica com `.map()`

## ▶️ Como executar

Clone o repositório:

```bash
git clone fidelmaluto/proreact

cd proreact

npm install

npm start
